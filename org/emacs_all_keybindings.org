| 按键            | 功能                                        |
|-----------------+---------------------------------------------|
| =?=               | 显示基本命令                                |
| =f=               | 搜索文件或目录                              |
|-----------------+---------------------------------------------|
| CRMD            | 拷贝、重命名、移动、删除(标记或当前)        |
| =;C=              | 拷贝                                        |
| =R=               | 重命名                                      |
| =D=               | 删除                                        |
| =;d=              | 标记删除                                    |
| =;x=              | 执行删除                                    |
| =;+/+=            | 创建目录                                    |
| =;=/==            | 比较两个文件                                |
| =;g=              | 反转当前buffer                              |
| =;k=              | 隐藏标记的文件 =C-r=  重新显示                |
|-----------------+---------------------------------------------|
| 子目录          |                                             |
| =I=               | 插入选中的目录                              |
| =gk=              | 移动到上一个子目录                          |
| =gj=              | 移到到下一个子目录                          |
|-----------------+---------------------------------------------|
| 标记            |                                             |
| =t=               | 标记/取消标记                               |
| =;m/C-SPC/TAB=    | 标记当前的并选择下一个                      |
| =;u=              | 取消标记                                    |
| =;U=              | 取消所有标记                                |
| =v=               | 反转所有标记                                |
| ="=               | 标记文件(正则)                              |
|-----------------+---------------------------------------------|
| 排序            | n/N 名称, e/E 扩展名,s/S大小                |
| =o?=              | o/O修改时间, c/C 创建时间, d 目录优先的最后 |
|-----------------+---------------------------------------------|
| 切换            |                                             |
| =i=               | 切换预览窗口(显示/隐藏 )                      |
| =z?=              | h-隐藏文件,(ifpP-+)...                      |
|-----------------+---------------------------------------------|
| shell           |                                             |
| =!=               | 在当前文件上执行shell 命令                  |
| =;&=              | 异步执行命令                                |
| =S=               | eshell 窗口                                 |
|-----------------+---------------------------------------------|
| 可修改模式      |                                             |
| =C-c C-e=         | 进入修改模式                                |
| =C-c C-c/C-c C-s= | 完成修改并退出编辑模式                      |
| =C-c C-k/C-c ESC= | 退出编辑模式并放弃所有修改                  |
| =C-x C-q=         | 退出编辑                                    |
|-----------------+---------------------------------------------|
| =du=              | 显示选中文件的大小                          |
| =q/ZZ=            | 退出                                        |
| =Q/ZQ=            | 退出                                        |
| =C-r=             | 刷新                                        |
| =k/j=             | 上下移动                                    |
| =h/-/left=        | 定位到父级目录                              |
| =l/RET/right=     | 进入选中目录                                |
| =gg=              | 到顶部                                      |
| =G=               | 到底部                                      |
| =<=               | 到上一个目录                                |
| =>=               | 到下一个目录                                |
| =gh=              | 到你的家目录($HOME)                         |
| =C-b=             | 往上翻一屏                                  |
| =C-f=             | 往下翻一屏                                  |
| =K/C-u=           | 往上翻半屏                                  |
| =J/C-d=           | 往下翻半屏                                  |
| =C-k=             | 向上滚动窗口                                |
| =C-j=             | 向下滚动窗口                                |
| =[=               | 上一个父级目录                              |
| =]=               | 下一个父级目录                              |
| =M-{=             | 上一个标记的文件/目录                       |
| =M-}=             | 下一个标记的文件/目录                       |
|                 |                                             |
