被引用的代码块:

function reactive(target) {
  console.log(target, '...')
}

文中引用者，语法 <<reactive>> :

function reactive(target) {
  console.log(target, '...')
}

// 调用 reactive
reactive({ a: 1 })

注意点：引用都需要指定 noweb 参数为 yes 。
